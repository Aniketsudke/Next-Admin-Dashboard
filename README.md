## Run Locally

Clone the project

```bash
  git clone https://github.com/Aniketsudke/Next-Admin-Dashboard.git
```

Go to the project directory

```bash
  cd Next-Admin-Dashboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
